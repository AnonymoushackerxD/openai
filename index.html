Subdomain Takeover Demonstration

Researcher: Bhavish Choudhary (AnonymoushackerxD)

Domain: promos.hoteltonight-test.com

Security Demonstration

Vulnerability: Subdomain Takeover via Fastly Misconfiguration

Status: Security Demonstration Only

Date: November 2025

Description

This subdomain promos.hoteltonight-test.com was vulnerable to takeover due to DNS pointing to Fastly without proper service configuration. This demonstration shows the security impact of such misconfigurations.

Technical Details

$ dig +short CNAME promos.hoteltonight-test.com
u2.shared.global.fastly.net.

$ curl -I https://promos.hoteltonight-test.com
HTTP/2 503
content-type: text/html
x-fastly-error: unknown domain

Impact

  • Brand reputation damage
  • Phishing attacks possibility
  • Malicious content hosting
  • Cookie stealing potential

Remediation

  1. Configure the domain in Fastly service
  2. Remove unused CNAME records
  3. Monitor subdomains regularly
  4. Use proper domain validation

This is a security demonstration page for educational purposes only. No actual systems were compromised in creating this demonstration.
